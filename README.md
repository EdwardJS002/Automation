# Installation d'une api SMS depuis un Raspberry Pi

### Systèmes d'Exploitation supportés:

- Raspberry PI OS 32 bits
- Raspberry PI OS 64 bits (beta)
- Ubuntu OS 32 bits
- Ubuntu OS 64 bits

### Commande

```bash
sh install.sh
```

### Brouillon

```bash
docker run -t -i --device=/dev/ttyUSB0 ubuntu bash
```
